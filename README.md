# Crypto Platform - Admin Dashboard
    
## Features
- KYC Review Center: Approve/Reject users
- User Management: Search, suspend users
- Analytics: AUM, volume, sign-ups
- Transaction Control: Approve withdrawals
- Plan Manager: Create staking plans
- Asset Config: Manage coins + fees
- Audit Logs

## Setup
1. `cd server && npm install && npm run dev`
2. `cd client && npm install && npm run dev`
3. Login: admin@crypto.com / admin123

Note: Admin-only build. Requires admin role JWT.
